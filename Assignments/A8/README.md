# Assignment 8 - Recipe Finder

## How to Run Locally

1. Navigate to the assignment folder:
```bash
cd Assignments/A8
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit:
```
http://localhost:5173
```

The app will automatically reload when you make changes.
